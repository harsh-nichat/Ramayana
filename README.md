# Old Archive: Ramayana 📜 (Legacy Project)

A frontend educational project built to explore the epic of Ramayana through an interactive web interface. This repository is maintained as a **code archive** to document my early foundations in web styling and structure.

## 🏛️ Project Overview
This project was designed to provide a multi-language (Hindi/English) experience, breaking down the various "Kands" (chapters) of the Ramayana with dedicated visual layouts.

## 🚀 Technical Features
- **Semantic HTML5:** Deep structure using multiple interconnected pages for different chapters.
- **Custom CSS:** Unique styling for each chapter to match the thematic elements of the story.
- **Bilingual Support:** Implementation of toggleable layouts for both English and Hindi text.
- **Asset Management:** Handling of multiple image assets and backgrounds to create an immersive UI.

## 📁 Key Components
- `index.html`: The main landing portal.
- `*_hindi.html`: Dedicated pages for Hindi localized content.
- `*.css`: Modular stylesheets for specific chapter themes (e.g., `balakand.css`, `yuddhakand.css`).

## 🛠️ Tech Stack
- **Language:** HTML5
- **Styling:** CSS3
- **Status:** Completed / Archived
